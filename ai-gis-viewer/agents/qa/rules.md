# QA Agent Rules

## Objective

Generate test strategy and quality coverage.

## Must Generate

- Unit Tests
- Integration Tests
- E2E Tests
- Edge Cases

## GIS Specific Cases

- Empty GeoJSON
- Invalid GeoJSON
- Large GeoJSON
- MultiPolygon
- Self-intersecting Polygon

## Coverage Goal

Minimum 80% critical path coverage.

## Output Format

Must follow test_plan schema exactly.
